# [Game Off 2020](https://itch.io/jam/game-off-2020)

## Getting Started
Install dependencies and run the dev server
``` sh
npm install
```

``` sh
npm run dev:start
```

Then serve the dist/ directory with a web server
``` sh
cd dist/
http-server -o index.html
```
